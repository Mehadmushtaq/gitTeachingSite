A quick Guide to how to use Git and GitHub

Here is simple tutorial of how to use GitHub and attach it to your projects. Just follow steps given below..

  1. Create New Folder and open it in vs Code.

  2. Open Terminal by Clicking on Terminal -> Open Terminal or you can use Ctrl+Shift+`.

    To initialize git use "git init" command

  3. Create New Repository on GitHub:
    • Give your repository a name and also can write description.
    • After that click on "Create Repository"

  4. Come back to VS Code add new file with name index.html.

    And write some code in it and save it by pressing Ctrl+S.

  5. Now Check status of git that will show "Modified". To see git status write ""git status" command

  6. Add changes to local repository. Write "git add . " command to do it.

    To make these changes permanent write command "git commit". You can also specify message in success scenerio that you want to show.
